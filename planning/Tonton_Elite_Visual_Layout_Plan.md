# Tonton Elite 网站素材布局与视觉方案

## 1. 首页 (Homepage) - 品牌感与信任感
首页的目标是让客户在 3 秒内相信你们是“专业工厂+大牌品质”。

| 位置 | 建议素材 | 视觉重点 |
| :--- | :--- | :--- |
| **Hero Banner (置顶海报)** | `案例合影` 中的大合照 | 展示多名学员穿着统一装备的场景，配文 "Forge Your Academy's Legacy"。 |
| **Our Partners (信任背书)** | 代工品牌 Logo 墙 (KNGZ, TATAMI等) | 放置在 Banner 下方，文字： "The Trusted Manufacturer for Global Elite Brands"。 |
| **About Our Craft (工艺展示)** | `生产视频/1.mp4` (剪辑成 5-10s 背景视频) | 背景循环播放热升华或四针六线缝纫镜头，体现工厂实力。 |
| **Featured Collections** | `产品高清图` 中的 4.png, 8.png | 干净的白底图，展示 MMA Shorts 和 Rash Guard 的标准版型。 |

---

## 2. 定制案例页 (Custom Gallery/Portfolio) - 真实感与参考
这是 B2B 客户停留时间最长的页面。

| 位置 | 建议素材 | 目的 |
| :--- | :--- | :--- |
| **Gallery Grid (瀑布流)** | `案例合影/` 文件夹内所有实拍图 | 展示不同设计风格（极简、全印花、团队 Logo 等）。 |
| **Success Stories** | 挑出 2-3 张最精致的合影 | 配上简短文字： "Custom Gear for [Gym Name], [Country]. 50+ sets delivered in 3 weeks." |

---

## 3. 产品详情页 (Product Page) - 专业参数与细节
让教练确信衣服不会在训练中撕裂。

| 位置 | 建议素材 | 描述重点 |
| :--- | :--- | :--- |
| **主图** | `产品高清图/` 中的正面、背面图 | 清晰的整体版型。 |
| **细节缩放图** | `产品高清图/` 中的缝线细节、腰部系带 | 强调四针六线 (4-needle 6-thread) 的平整度和腰部防滑设计。 |
| **功能视频** | `生产视频/` 中的拉伸测试片段 | 展示 4-way stretch (四向弹力) 的回弹效果。 |

---

## 4. 后续操作建议：Shopify 快速启动
1. **域名注册**：建议尽快手动查验并注册 `tontonelite.com` (如已被占用，可选 `tontonelite-custom.com`)。
2. **主题选择**：推荐 Shopify 官方主题 **"Dawn"** (免费且简洁，适合展示图片) 或付费主题 **"Empire"** (适合展示大量 B2B 产品目录)。
3. **插件安装**：
    - **Globo Request a Quote**：用于收集询盘（不直接结账）。
    - **Quantity Breaks**：展示阶梯定价 (10-50, 50-100等)。
